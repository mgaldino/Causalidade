# Codebook - Obra Transparente DiD Analysis

**Date**: 2026-02-11
**Project**: Replication package for "The Effect of Civil Society Oversight on Public Service Delivery: A Mixed-Methods Study of Construction Monitoring in Brazil" (Galdino, Mondo, Sakai, and Paiva)

---

## Data Overview

This replication package contains data for a difference-in-differences (DiD) analysis of the Obra Transparente project's impact on public school and nursery construction completion rates in Brazil.

### Data Sources

| Source | Description | Access |
|--------|-------------|--------|
| SIMEC/FNDE | Construction monitoring system | Snapshots via Transparência Brasil |
| IBGE/IPEA | Municipal socioeconomic data | Atlas Brasil |
| TSE | Electoral data | Public data portal |
| Transparência Brasil | List of treated works | Project records |

### Study Period

| Period | Date | `periodo` | `rel_time` | Phase |
|--------|------|-----------|------------|-------|
| 1 | Aug 2015 | 0 | -3 | Pre-treatment |
| 2 | May 2017 | 1 | -2 | Pre-treatment |
| 3 | Mar 2018 | 2 | -1 | Pre-treatment (reference) |
| 4 | Sep 2018 | 3 | 0 | Treatment starts |
| 5 | Aug 2019 | 4 | +1 | Post-treatment |
| 6 | Oct 2023 | 5 | +2 | Long-term follow-up |

### Geographic Coverage

States included: SP, MG, SC, PR, RS (South and Southeast Brazil)

## Processed Data (`data/processed/`)

These files are generated by the analysis scripts.

### `did_panel_6periods.rds` (MAIN ANALYSIS FILE)

**Description**: Balanced panel for DiD analysis with 6 periods.

**Created by**: `code/01_did_analysis_6periods.R`

**Dimensions**: 20,964 rows × 9 columns

**Unit of observation**: Construction project × Period

| Variable | Type | Description | Values |
|----------|------|-------------|--------|
| `id` | numeric | Unique project ID in SIMEC | e.g., 1366 |
| `concluida` | numeric | Project completed in this period | 0 = No, 1 = Yes |
| `municipio` | character | Municipality name | e.g., "Conchas" |
| `uf` | character | State abbreviation | SP, MG, SC, PR, RS |
| `group_treated` | numeric | Municipality in treated group | 0 = Control, 1 = Treated |
| `periodo` | numeric | Panel period (0-indexed) | 0, 1, 2, 3, 4, 5 |
| `post` | numeric | Post-treatment indicator (periodo >= 3) | 0 or 1 |
| `treat_post` | numeric | Treatment × Post interaction | 0 or 1 |
| `rel_time` | numeric | Time relative to treatment onset | -3, -2, -1, 0, +1, +2 |

**How `concluida` is computed**:
```r
concluida <- as.numeric(grepl("Concluída", situacao))
```

**How `group_treated` is computed**:
```r
# Based on whether the municipality has at least one work in the treated list
group_treated <- max(indicator_ot)  # per municipality
```

### `did_results_6periods.rds`

**Description**: Complete DiD analysis results.

**Content** (R list):
- `panel`: Complete dataset
- `stats`: Descriptive statistics
- `model_twfe`: TWFE model (static DiD)
- `model_es`: Event study model
- `es_coefs`: Event study coefficients

## Reproducibility Notes

### Reproducible Pipeline

The complete panel is built from traceable sources:
1. `.Rdata` files from Transparência Brasil
2. CSVs downloaded automatically from GitHub
3. Excel list of treated works

### Balanced Panel

The panel uses only construction projects present across all 6 periods to ensure balance. Projects that appear only in later periods are excluded.

### Variable `concluida`

Computed using `grepl("Concluída", situacao)` applied to the `situacao` field in each SIMEC snapshot.
